# MatrixCalculator 
微信小程序-矩阵计算工具  
### 项目简述
本小程序提供基本的矩阵运算。在尽量降低用户学习成本的同时，矩阵输入方式效率较高。  
本小程序使用基于Vue.js的uni-app进行开发，现已上架微信，名为“矩阵计算工具”。
